Nachdem mein Management-Nuc vorbereitet ist, folgt nun der NSX-Manager. Zunächst erstelle ich den DNS-Eintrag einschließlich des Reverse-Eintrags. Danach erfolgt die Installation des NSX-Managers.

Rechtsklick auf den vCenter-Cluster -> „Deploy OVF Template“.

![NSX Manager 1](/assets/2024-11-10-Homelab_1.3/nsx-manager-1.png)

Wählen Sie im Bildschirm "Select an OVF Template" die NSX Unified Appliance OVA-Datei aus.

![NSX Manager 2](/assets/2024-11-10-Homelab_1.3/nsx-manager-2.png)

Wählen Sie im Bildschirm "Select a name and folder" das Ziel-Datacenter-Objekt aus.

![NSX Manager 3](/assets/2024-11-10-Homelab_1.3/nsx-manager-3.png)

Wählen Sie im Bildschirm "Select a compute resource" den Ziel-vCenter-Cluster aus.

![NSX Manager 4](/assets/2024-11-10-Homelab_1.3/nsx-manager-4.png)
![NSX Manager 5](/assets/2024-11-10-Homelab_1.3/nsx-manager-5.png)

Wählen Sie im Bildschirm "Configuration" mindestens die Medium-Konfiguration aus.

![NSX Manager 6](/assets/2024-11-10-Homelab_1.3/nsx-manager-6.png)

Wählen Sie im Bildschirm "Select storage" "Thin Provision" und das gewünschte Datastore aus.

![NSX Manager 7](/assets/2024-11-10-Homelab_1.3/nsx-manager-7.png)

Geben Sie für Network1 das VLAN-Management-Netzwerk ein.

![NSX Manager 8](/assets/2024-11-10-Homelab_1.3/nsx-manager-8.png)

Geben Sie starke Passwörter für alle Benutzertypen ein.

![NSX Manager 9](/assets/2024-11-10-Homelab_1.3/nsx-manager-9.png)

Geben Sie den Hostnamen ein, z. B. nsx-manager.  
Geben Sie den Rollennamen ein, z. B. NSX Manager.  
Geben Sie die Gateway-IP-Adresse ein, z. B. 10.173.62.253.  
Geben Sie eine öffentliche IP-Adresse für die VM ein, z. B. 10.173.62.44.  
Geben Sie die Netzmaske ein, z. B. 255.255.255.0.

![NSX Manager 10](/assets/2024-11-10-Homelab_1.3/nsx-manager-10.png)

Geben Sie den DNS-Server ein, z. B. 10.172.40.1.  
Geben Sie den NTP-Server ein, z. B. 10.113.60.176.  
Aktivieren Sie das Kontrollkästchen "Enable SSH".  
Aktivieren Sie das Kontrollkästchen "Allow SSH root logins".

![NSX Manager 11](/assets/2024-11-10-Homelab_1.3/nsx-manager-11.png)

Klicken Sie auf "Finish", und NSX-T Manager 1 wird mit der Bereitstellung gestartet.

![NSX Manager 12](/assets/2024-11-10-Homelab_1.3/nsx-manager-12.png)

Wenn die Bereitstellung abgeschlossen ist, wählen Sie die VM aus und schalten Sie sie ein.  
Greifen Sie auf die Web-Konsole von NSX-T Manager 1 zu, indem Sie die URL aufrufen, z. B.: https://10.173.62.44/.

![NSX Manager 13](/assets/2024-11-10-Homelab_1.3/nsx-manager-13.png)
